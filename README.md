# Movie Correlation Analysis (Python)

## 📊 Overview

This project analyzes movie data to identify which factors most strongly influence box office success.

Using Python, the analysis explores relationships between variables such as budget, votes, ratings, and gross revenue.

---

## 🎯 Objectives

* Identify correlations between movie features and revenue
* Determine which variables most impact financial success
* Visualize relationships using charts and heatmaps

---

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy)
* Matplotlib / Seaborn (Visualization)
* Jupyter Notebook

---

## 📁 Dataset

* Movie dataset including budget, gross revenue, ratings, votes, and genre

---

## 🔍 Analysis

### 1. Data Cleaning

* Removed missing values
* Standardized data types
* Filtered relevant columns

### 2. Correlation Matrix

* Generated correlation heatmap to identify relationships between variables

### 3. Key Comparisons

* Budget vs Gross Revenue
* Votes vs Gross Revenue
* Ratings vs Gross Revenue

---

## 📈 Key Insights

* Budget showed a strong positive correlation with gross revenue
* Number of votes also correlated with higher revenue
* Ratings had a weaker correlation than expected
